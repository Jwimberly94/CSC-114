Q : what is a string? 


3:23 PM
A string is a sequence of characters, like text. In Python you create one by enclosing characters in quote marks—either single (') or double (") quotes.

python
print('Python programming is fun!')
print("Cortana")
Both work the same way. Strings are stored using the str data type. If you need a string to contain an apostrophe or a quote mark itself, use the other type of quote to enclose it—like "Don't fear!" or 'She said "fire!"'. You can also use triple quotes (""" or ''') if you need both types of quotes in one string, or if you want the string to span multiple lines.
